# 면책조항

## 개인 프로젝트

이 프로젝트는 이기쁨(Kipeum Lee)의 독립적인 개인 프로젝트입니다. 어떠한 법률 실무 조직, 고용주 또는 조직의 후원, 보증 또는 제휴를 받지 않습니다. 모든 의견, 설계 결정 및 콘텐츠는 전적으로 본인의 것입니다.

## 법률 자문이 아닙니다

이 저장소의 어떠한 내용 — 리뷰 코멘트, 레드라인 마크업, 커버 메모, 스코어카드 평가, 릴리스 권고, 기타 에이전트가 생성한 모든 산출물을 포함하여 — 도 법률 자문을 구성하지 않습니다. 산출물은 정보 제공 및 보조 목적에 한합니다. 자격을 갖춘 법률 전문가의 자문을 대체하는 용도로 사용하지 마세요.

## AI의 한계 — 환각(Hallucination) 및 오류

이 프로젝트는 대규모 언어 모델(Claude)을 기반으로 합니다. LLM은 **환각**, **사실 오류**, **부정확한 법률 분석**을 생성할 수 있으며, 실제로 생성합니다. 산출물에는 다음과 같은 문제가 포함될 수 있습니다:

- **검증 결과 날조** — 실제로 확인하지 않은 인용을 검증 완료로 보고하거나, 실존하는 인용을 부존재로 보고
- **인용 상태 오분류** — 주장을 뒷받침하지 않는 법령에 `Verified`를 부여하거나, 유효하지만 생소한 인용에 `Nonexistent`를 부여
- **핵심 이슈 누락** — 검토 대상 문서의 환각 인용, 논리적 결함 또는 실체적 오류를 식별하지 못함
- **거짓 양성(False Positive) 생성** — 올바른 인용, 건전한 추론 또는 적절한 서식을 결함으로 지적
- **비일관적 점수 산정** — 실제 findings와 일치하지 않는 등급 또는 릴리스 권고를 부여
- **리뷰 코멘트 내 환각 도입** — 여백 코멘트에 그 자체로 근거가 없거나 부정확한 사실 주장을 기재
- **문서 구조 오파싱** — 복잡한 DOCX 파일에서 섹션, 번호 체계, 교차참조 또는 정의 용어를 올바르게 추출하지 못함

에이전트에는 내장 안전장치(이중 트랙 인용 검증, 검증 상태 분류 체계, 7항목 자체 검증 품질 게이트, 반환각 지침)가 포함되어 있지만, **이러한 안전장치는 오류의 위험을 줄일 뿐 제거하지는 않습니다**. 자격을 갖춘 사람이 모든 산출물을 신뢰하거나 조치를 취하기 전에 반드시 검토하고 검증해야 합니다. 에이전트는 보조하고, 사람이 판단합니다.

## 데이터 처리

에이전트는 LLM API 호출을 수행하며, 이는 문서 내용, 리뷰 findings 및 검색 쿼리가 모델 제공자(Anthropic)에게 전송됨을 의미합니다. 표준 상업 이용약관에 따라 API 요청 수명 주기를 넘어 외부 서버에 데이터가 저장되지 않습니다.

기밀 또는 특권(privilege) 정보가 포함된 사안에 이 도구를 사용하는 경우:

- 적절한 데이터 처리 약관이 포함된 상업용 API 키를 사용하세요.
- 최신 약관은 [Anthropic의 데이터 사용 정책](https://www.anthropic.com/policies)을 확인하세요.
- 민감한 자료를 AI 도구로 처리하기 전에 소속 조직의 보안 및 컴플라이언스 요구사항을 확인하세요.
- 외부 API를 통해 처리되는 변호사-의뢰인 특권 문서는 일부 관할권에서 특권 포기(waiver of privilege)의 위험이 있을 수 있음을 인지하세요.

## 책임 범위

저자는 다음에 대해 책임을 지지 않습니다:

- 에이전트의 리뷰 산출물에 기반한 모든 결정
- 생성된 리뷰 findings, 점수 또는 릴리스 권고에 대한 신뢰로 인한 손실, 법적 결과 또는 손해
- 리뷰 산출물의 정확성, 완전성 또는 특정 목적에의 적합성
- "Pass" 권고에 기반하여 발송되었으나 이후 오류가 발견된 문서
- "Release Not Recommended" finding이 거짓 양성(false positive)이었던 경우 보류된 문서
- 잘못 구성된 API 키, 네트워크 설정 또는 사용자 오류로 인한 데이터 노출

**이 프로젝트를 사용함으로써 귀하는 이 면책조항을 읽고 이해했음을 인정합니다.**
